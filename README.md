# 📚 Sistema de Cadastro de Alunos

Este projeto foi desenvolvido como parte do meu processo de aprendizado em **Python**, com foco na prática de operações de **CRUD** (Create, Read, Update, Delete), integração com banco de dados **SQLite** e criação de **interfaces gráficas utilizando Tkinter**.

## 🎯 Objetivo

O principal objetivo do projeto é aplicar conceitos de:

- Programação em Python
- Desenvolvimento de interfaces gráficas com Tkinter
- Manipulação de banco de dados local com SQLite
- Operações CRUD
- Organização e estruturação de código

## 🛠️ Tecnologias Utilizadas

- Python
- SQLite (banco de dados local)
- Tkinter (interface gráfica)

## ⚙️ Funcionalidades

O sistema permite:

- ✅ Cadastrar alunos
- 📋 Listar alunos cadastrados
- ✏️ Atualizar informações de alunos
- ❌ Excluir alunos do banco de dados

Todas as operações são realizadas através de uma **interface gráfica desenvolvida com Tkinter**, facilitando a interação do usuário com o sistema.

## 🗄️ Banco de Dados

O projeto utiliza **SQLite** como banco de dados local, permitindo armazenar os dados dos alunos sem a necessidade de instalar um servidor de banco de dados externo. O banco é criado automaticamente caso não exista.
